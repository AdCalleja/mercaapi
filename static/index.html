mercaapi - Unofficial Mercadona Products API

mercaapi is an unofficial API that provides access to Mercadona product data, enhanced with additional nutritional information. This project aims to offer developers and researchers easy access to comprehensive product data from Mercadona, including detailed nutritional facts not readily available through official channels.

Features

  • Comprehensive product data from Mercadona
  • Enhanced nutritional information for each product
  • RESTful API endpoints for easy integration
  • Docker support for simple deployment
  • Regular updates to keep product information current
